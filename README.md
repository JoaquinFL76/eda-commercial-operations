# EDA Commercial Operations

Business-oriented exploratory data analysis project focused on commercial activity, sales performance and operational insights.

## Project purpose

This repository shows how exploratory data analysis can help understand business performance, detect patterns and support better decision-making.

The goal is not only to visualize data, but to connect analysis with practical business questions such as:

- Which customers or segments generate more revenue?
- What variables may be affecting sales performance?
- Are there patterns by region, product, channel or period?
- Where do we see anomalies, concentration or missed opportunities?
- What operational insights could improve commercial follow-up?

## Approach

This project follows a practical EDA workflow:

1. Data loading and initial inspection
2. Data cleaning and preparation
3. Descriptive statistics
4. Univariate and bivariate analysis
5. Visualization of trends, distributions and relationships
6. Business interpretation of the findings
7. Summary of insights and possible next steps

## Tools used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
